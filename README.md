# Glendar

Glendar is FUCKIN AWESOME BEAUTIFUL SwordWorld 2.0 Online Session Tool.

## Requirements

* node => 7.0
* 諦めない心 => 1.0

## Installation

1. Make firebase application.

https://console.firebase.google.com/

2. Write dotenv file.

```dotenv
FIREBASE_APIKEY=
FIREBASE_DOMAIN=
FIREBASE_DATABASE_URL=
FIREBASE_BUCKET=
FIREBASE_SENDER=
DICEAPI_ROOT=
```

3. Launch Application.

```
$ npm i
$ npm start
```
